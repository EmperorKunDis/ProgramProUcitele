Výběr Modelu

Antigravity vám dává možnost vybrat si mozek, který pohání vaše agenty. Pro různé úkoly se hodí různé modely.

Dostupné Modely

Gemini 3 Pro (Doporučeno)

Zlatý standard. Nejlepší poměr mezi cenou, rychlostí a schopnostmi. Používá se jako defaultní model pro většinu úkolů (Executive & Coding).

Gemini 3 Flash

Extrémně rychlý a levný model. Ideální pro:

  • Rychlé opravy chyb (Quick Fix)
  • Generování commit zpráv
  • Jednoduché refaktoringy

Claude 3.7 Sonnet (Volitelné)

Pokud preferujete modely od Anthropic, Antigravity je podporuje prostřednictvím API klíče. Vynikající na psaní dokumentace a kreativní psaní.

Lokální Modely

Pro práci s citlivými daty můžete použít modely běžící přímo na vašem stroji (vyžaduje 32GB+ RAM).

  • Gemma 2 (27B)
  • Llama 3 (70B)

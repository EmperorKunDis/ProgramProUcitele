První Spuštění

Po dokončení Setup Wizardu je čas spustit Antigravity a seznámit se s rozhraním.

Otevření Projektu

Spusťte Antigravity a vyberte způsob otevření:

  • Open Folder - Otevře existující projekt
  • Clone Repository - Naklonuje Git repo
  • Playground - Prázdný sandbox pro experimenty

Z terminálu

# Otevře aktuální složku
agy .

# Otevře konkrétní projekt
agy ~/projekty/muj-web

Orientace v Rozhraní

Antigravity má dvě hlavní "views":

1. Agent Manager

Hlavní obrazovka pro práci s AI agentem. Zde:

  • Zadáváte úkoly přirozeným jazykem
  • Vidíte historii konverzací (Inbox)
  • Spravujete workspaces

2. Editor View

Klasické IDE rozhraní známé z VS Code:

  • File Explorer vlevo
  • Editor uprostřed
  • Terminal dole
  • Agent panel vpravo

Klávesové zkratky

Zkratka Akce
Ctrl + ` Toggle Terminal panel
Cmd + L Toggle Agent panel
Cmd + Shift + P Command Palette
Cmd + K Inline chat (v editoru)

První Konverzace

Vyzkoušejte agenta jednoduchým úkolem:

Vytvoř soubor hello.py s funkcí, která pozdraví uživatele jménem.

Agent by měl:

  1. Porozumět požadavku
  2. Vytvořit soubor hello.py
  3. Implementovat funkci
  4. Případně nabídnout spuštění
✅
Tip: Pokud agent udělal přesně to, co jste chtěli, napište Looks good nebo OK. Pokud ne, popište co změnit.

Nastavení Extensions

Pro váš programovací jazyk možná budete chtít nainstalovat rozšíření:

  1. Klikněte na ikonu Extensions (čtvereček) v levém panelu
  2. Vyhledejte rozšíření (např. "Python", "ESLint")
  3. Klikněte Install
ℹ️
Antigravity používá Open VSX marketplace (ne VS Code Marketplace). Většina populárních extensions je dostupná.

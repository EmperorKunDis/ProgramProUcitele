První Spuštění

Po úspěšné instalaci vás čeká úvodní nastavení (Onboarding Wizard). Tento proces nastaví vaše prostředí pro optimální spolupráci s AI.

✨
Screenshot: Welcome Wizard
Úvodní obrazovka s logem Antigravity/PRAUT

1. Přihlášení

Klikněte na Sign in with Google. Musíte použít účet, který má povolený přístup k Gemini API (nebo firemní PRAUT účet).

2. Výběr Motivu

Antigravity respektuje nastavení vašeho systému, ale můžete si vynutit:

  • Dark Mode (Doporučeno pro vývojáře)
  • Light Mode
  • High Contrast

3. Import Nastavení

Pokud přecházíte z VS Code, Antigravity může importovat vaše:

  • Klávesové zkratky (Keybindings)
  • Snippety
  • Některá rozšíření (Extensions)
ℹ️
Rozšíření: Ne všechna VS Code rozšíření jsou kompatibilní. AI agenti často nahrazují potřebu mnoha pluginů (např. pro formátování nebo linting).

4. Konfigurace Agenta

V posledním kroku si pojmenujete svého primárního agenta. Můžete mu dát jméno (např. Jarvis, Friday, nebo české Karel).

Zde také nastavujete úroveň autonomie:

  • Konzervativní: Agent se ptá před každou změnou souboru.
  • Vyvážená (Default): Agent se ptá u kritických změn a mazání.
  • Autonomní: Agent pracuje rychle, ptá se jen při nejasnostech.
